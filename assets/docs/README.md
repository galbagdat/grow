# Папка для PDF программы

Сюда нужно положить файл **`icu-program.pdf`** — на него ссылается кнопка «Скачать программу (PDF)» в Блоке 3 главной страницы.

## Как заменить

1. Подготовьте финальную программу обучения в PDF (название: **`icu-program.pdf`**, без пробелов и кириллицы)
2. Положите файл в эту папку: `assets/docs/icu-program.pdf`
3. Сделайте commit и push в `main` — файл автоматически попадёт на GitHub Pages
4. Кнопка «Скачать программу (PDF)» на сайте начнёт работать

## Рекомендации к файлу

- **Имя файла:** только латиница, без пробелов — `icu-program.pdf` (или, если нужно версионировать: `icu-program-v2.pdf`, но тогда поправить ссылку в [`index.html`](../../index.html))
- **Размер:** до 5 МБ — иначе мобильные пользователи будут долго ждать загрузку
- **Структура внутри PDF:**
  - Обложка с лого ICU Kazakhstan
  - Краткое описание программы (1 стр.)
  - Каждый из 8 модулей: название, длительность, формат, результат для студента
  - Условия сертификации
  - Информация о Diploma Supplement европейского образца
  - Контакты для записи (WhatsApp + email)
- **PDF/A** или PDF с встроенными шрифтами — чтобы корректно открывался на любых устройствах

## Где ссылка используется

Текущая ссылка на PDF — в нескольких местах:

- [`index.html`](../../index.html) → Блок 3 → кнопка «Скачать программу (PDF)»
- Возможно появится в WhatsApp-боте (горячий трек, шаг 2 PRD §4.4) — там бот будет отправлять этот же файл по запросу

## Если поменяли имя файла

Замените все упоминания `icu-program.pdf` в проекте — в [`index.html`](../../index.html) и (на этапе разработки бота) в шаблонах WhatsApp.
